## Articoli determinativi

Si usano quando il riferimento è **determinato**: **già noto**, **univoco** nel contesto o **identificabile**.

Per il **maschile**, _s + consonante_ indica la **s impura** (es. *st-*, *sc-*): *il sole*, non *lo sole*. La stessa suddivisione sull’**inizio** della parola vale anche per **uno** e per l’**articolo partitivo** nelle tabelle sotto.

| Articoli determinativi | Singolare | Plurale |
| ---------------------- | --------- | ------- |
| **Maschile**           |           |         |
| Nome che inizia per **z**, **gn**, **pn**, **ps**, **x**, **y** o **s** + consonante | **lo** | **gli** |
| Nome che inizia per **vocale** | **l’** | **gli** |
| Nome che inizia per **altra consonante** | **il** | **i** |
| **Femminile**          |           |         |
| Nome che inizia per **consonante** | **la** | **le** |
| Nome che inizia per **vocale** | **l’** | **le** |

## Articoli indeterminativi

Indicano qualcosa di **non specificato** o **generico**, **non ancora identificato** nel discorso. Forme: *un*, *uno*, *una*, *un’*.

| Articoli indeterminativi | Singolare |
| ------------------------ | --------- |
| **Maschile**             |           |
| Nome che inizia per **z**, **gn**, **pn**, **ps**, **x**, **y** o **s** + consonante | **uno** |
| Nome che inizia per **vocale** o **altra consonante** | **un** |
| **Femminile**            |           |
| Nome che inizia per **consonante** | **una** |
| Nome che inizia per **vocale** | **un’** |

_Nota:_ esistono solo al **singolare**. Al plurale: **articolo partitivo** o **indefiniti di quantità** (non articoli), es. _dei / delle_, _alcuni / alcune_.

## Articolo partitivo

Serve per una **parte** o **quantità non precisata**, spesso con nomi **non numerabili**. *Un po’ di* esprime quantità ma **non** è *di* + articolo.

La **forma** segue **genere**, **numero** e, al maschile, l’**inizio** del nome come nella tabella dei determinativi.

| Articoli partitivi | Singolare | Plurale |
| ------------------ | --------- | ------- |
| **Maschile**       |           |         |
| Nome che inizia per **z**, **gn**, **pn**, **ps**, **x**, **y** o **s** + consonante | **dello** | **degli** |
| Nome che inizia per **vocale** | **dell’** | **degli** |
| Nome che inizia per **altra consonante** | **del** | **dei** |
| **Femminile**      |           |         |
| Nome che inizia per **consonante** | **della** | **delle** |
| Nome che inizia per **vocale** | **dell’** | **delle** |

_Esempi:_ _Voglio dell’acqua._ _Hanno dei figli._ _Ci sono delle mele._

## Gli articoli si omettono

1. **Quando si vuole conferire un senso d’indeterminatezza** — *Avete bibite fresche?*
2. **In alcune locuzioni avverbiali** come *a ragione*, *in sostanza*, *di fretta*, *di proposito*, *a spasso*, *di corsa*, *per pietà*, *a dirotto*, ecc. — *Roberto e Pina vanno a spasso nel parco.*
3. **In alcune locuzioni verbali** come *parlare italiano*, *avere fame*, *avere sete*, *avere sonno*, *cercare casa*, *cercare lavoro*, ecc. — *Il neonato piange perché ha fame.*
4. **In locuzioni in forma di sentenza, proverbio o modo di dire** — *A buon intenditor poche parole.*
5. **In messaggi brevi** come telegrammi, annunci economici e insegne — *Cerco appartamento zona centro.*
6. **In elenchi o enumerazioni** — *Le tabaccherie vendono sigarette, francobolli, caramelle, articoli di cancelleria*, ecc.
7. **A volte nei titoli** dei libri e dei giornali — *Grammatica italiana per stranieri.*
8. **Con i nomi dei mesi, dei giorni e delle stagioni** in assenza di specificazione — *Una rondine non fa primavera.* Con la specificazione si usa l’articolo: *La Primavera di Praga ebbe inizio nel 1968.*
9. **Quando c’è un nome seguito da un aggettivo possessivo** — *Elena parcheggia sempre davanti a casa sua.*
10. **Nelle espressioni introdotte da *come*** (in qualità di) — *Come studenti si ha diritto allo sconto sui biglietti del cinema.*
11. **In alcuni complementi di luogo** come *a casa*, *in vacanza*, *in chiesa*, *a teatro*, *in ufficio*, *in città*, *in campagna*, *per mare*, *a tavola*, *in lungo e in largo*, ecc. — *L’ho cercata ovunque, per mare e per terra.*
12. **Nel complemento di materia** — *Ho ricevuto in dono un orologio d’oro.*
13. **Complemento d’argomento** — in assenza di specificazione: *Questo è un libro di storia.* Con la specificazione: *Questo è un libro sulla storia del Risorgimento italiano.*
14. **Espressioni di tempo** — *Mio fratello lavora da mattina a sera.* *Alessandro ieri è rincasato a notte fonda.*
15. **Espressioni di causa, mezzo, scopo e modo** — es. *morire di*, *tremare di*, *a piedi*, *in bicicletta*, *a memoria*, *di corsa*, *a maniche lunghe*, ecc. — *Ho comprato ai grandi magazzini una borsa da viaggio.*
16. **Espressioni di quantità** — dopo *più*, *meno*, *abbastanza*: *Oggi fa abbastanza caldo.*
17. **Espressioni con *con* e *senza*** — in assenza di specificazione: *Sofia va a scuola di danza con grande entusiasmo.* Con la specificazione: *Patrizio prende parte alla scampagnata con l’entusiasmo di un bambino.*
18. **Possessivi predicativi** — *È nostra intenzione svegliarci prestissimo domani mattina.*

## Preposizioni + articolo (richiamo)

Quando **di, a, da, in, su** (e a volte **con, per, tra/fra**) si uniscono all’articolo determinativo, si formano le **preposizioni articolate** (_al, dal, nel, sul, nella…_). Seguono le stesse regole di **il / lo / l’** ecc.; il dettaglio è in `preposizioni.md`.
